# Stream API 和 Lambda 運算式

Stream API 是一種處理集合資料(DataSet)的方式，允許用宣告式的方式對集合資料進行各種操作，不改變原始資料結構，而是返回新的結果。通常比傳統的迴圈更簡潔且易於理解。

Lambda 運算式是簡潔的匿名函數寫法，可作為 Stream API 中許多操作的參數，用來定義具體的處理邏輯。

範例:
- `filter(n -> n % 2 == 0)` → 只保留偶數
- `map(n -> n * n)` → 每個數字平方
- `reduce(0, Integer::sum)` → 累加計算總和
- `sorted()` → 升序排序
