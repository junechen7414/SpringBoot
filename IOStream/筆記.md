# InputStream 與 OutputStream

## InputStream

*   從資料來源如檔案讀取位元組資料。
*   是所有輸入串流的抽象基底類別。
*   提供多種方法來讀取資料，例如：
    *   `read()`：讀取單個位元組。
    *   `read(byte[] b)`：讀取多個位元組到緩衝區。
    *   `read(byte[] b, int off, int len)`：從指定的偏移量開始，讀取指定長度的位元組到緩衝區。

## OutputStream

*   將位元組資料寫入到目標如檔案。
*   是所有輸出串流的抽象基底類別。
*   提供多種方法來寫入資料，例如：
    *   `write(int b)`：寫入單個位元組。
    *   `write(byte[] b)`：寫入多個位元組從緩衝區。
    *   `write(byte[] b, int off, int len)`：從指定的偏移量開始，寫入指定長度的位元組從緩衝區。

*   範例 :
    *   InputStream 與 FileInputStream：用於從檔案中讀取位元組資料。

    *   OutputStream 與 FileOutputStream：用於將位元組資料寫入檔案。

    *   try-with-resources：確保在程式執行完畢後，自動關閉開啟的檔案串流，避免資源浪費。

    *   buffer：透過緩衝區一次讀取多個位元組，提高效能。
